CURRENT .zshrc SECTIONS
========================
1. bus0()      — Quick direct shell on bus-home: sets terminal title "bus-home", arista-ssh check-auth || login, then ssh bus-home
   busclaude() — Like bus0 but opens Claude Code: sets title "bus-claude", auth, then ssh -t bus-home 'bash -l -c "cd ~/claude && ./lclaude"'
                 bash -l required: non-login SSH shells skip ~/.bashrc so ~/.local/bin (claude binary) is missing from PATH
   bus()       — Interactive tmux session menu for jump host (bus-home) via SSH multiplexing + expect
2. wsc()  — Remote Wireshark live capture via SSH + named pipe (FIFO) to Wireshark app
           NOTE: ssh_error uses 2>&1 >/dev/null to capture stderr for host key change detection
3. General Settings + PATH + CDPATH:
   - ~/go/bin prepended to PATH (fzf, gnmi, and other go-installed binaries)
   - fzf keybindings via source <(fzf --zsh) — Ctrl+R history, Ctrl+T file, Alt+C dir
   - Up arrow bound to fzf-history-widget (^[[A + ^[OA) immediately after source <(fzf --zsh)
     so the widget exists before bindkey runs; matches bus-home up arrow behaviour
   - FZF_DEFAULT_OPTS: --height 30% --layout=reverse --border
   - FZF_TMUX_HEIGHT=30% (controls fzf internal default height — must be set)
   - FZF_CTRL_R_OPTS: --height 30% --no-sort --exact (force height override, preserve order)
   - FZF_CTRL_T_OPTS: bat preview right:50% (Ctrl+T file search with syntax-highlighted preview)
   - FZF_ALT_C_OPTS: tree fallback ls preview right:40% (Alt+C dir jump with tree or ls preview)
   - Up arrow: fzf-history-widget (both ^[[A and ^[OA); Ctrl+R also works
   (fzf installed via: go install github.com/junegunn/fzf@latest)
   - zsh-autosuggestions: cloned to ~/.zsh/zsh-autosuggestions/, sourced after fzf
     ZSH_AUTOSUGGEST_HIGHLIGHT_STYLE=fg=cyan set in General Settings
   - stty -ixon                              (disable Ctrl+S/Q flow control)
   - setopt IGNORE_EOF + IGNOREEOF=2         (require 3 Ctrl+D presses to exit)
   - TZ=Asia/Kolkata                         (timezone)
   - ZSH_AUTOSUGGEST_HIGHLIGHT_STYLE=fg=cyan (autosuggestion color, visible in tmux)
   - EDITOR/VISUAL=vim
4. History: HISTSIZE=100000, SAVEHIST=200000, SHARE_HISTORY (syncs across tmux panes, implies HIST_APPEND),
            HIST_IGNORE_ALL_DUPS (dedup; supersedes HIST_IGNORE_DUPS — not set),
            HIST_FIND_NO_DUPS (up/down arrow skips duplicates)
5. Aliases: ls/ll/la/lrtha use GNU --color=auto if available, else BSD -G (auto-detected at startup); grep --color=auto; .., ..., mkcd()
6. Prompt: green user@host, blue cwd, git branch via vcs_info
   NOTE: uses precmd_functions+=(vcs_info_precmd) NOT precmd() directly — avoids conflict with iTerm2
7. iTerm2 shell integration: sourced AFTER prompt so it appends to precmd_functions cleanly
   Skips itself automatically inside tmux ($TERM == tmux-256color)
