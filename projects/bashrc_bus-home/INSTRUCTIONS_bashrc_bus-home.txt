CURRENT .bashrc SECTIONS
========================
0.  ble.sh — source ~/.local/share/blesh/ble.sh --noattach (interactive only; silent if missing)
    MUST be first. Loaded in noattach mode so all other sections run before line-editing takeover.
1.  Load System Defaults (/etc/bashrc 2>/dev/null) + set -h to re-enable hashing
    /etc/bashrc disables hashing (set +h); re-enabling is required for NVM and other tools
2.  Fix Tmux Resizing (shopt -s checkwinsize)
3.  Standard Tab Completion (bash-completion)
4.  FZF: source ~/.fzf.bash, FZF_TMUX_HEIGHT=30%, FZF_DEFAULT_OPTS (30% height, reverse, border),
    FZF_CTRL_R_OPTS (--height 30%, no-sort, exact),
    FZF_CTRL_T_OPTS (bat preview right:50%, file search via Ctrl+T),
    FZF_ALT_C_OPTS (ls preview right:40%, dir jump via Alt+C);
    bat available, fd/tree not installed
5.  Custom Functions — command_not_found_handle (@<device> SSHes to admin@<device>); mkcd()
    IMPORTANT: must stay BEFORE section 13 (ble-attach) so ble.sh wraps the handler on load
6.  Prevent Ctrl+D (ignoreeof, IGNOREEOF=10)
7.  Aliases: ta, fix-tmux, tmux-studio, ll, la, ls, lrtha, grep, .., ..., eod
8.  TZ=Asia/Kolkata, PATH prepend ~/.local/bin:~/.atuin/bin, EDITOR/VISUAL=vim, CDPATH
9.  History: HISTSIZE=100000, HISTFILESIZE=200000, ignoredups:erasedups, histappend
    PROMPT_COMMAND: guarded with [[ != *history -a* ]] to prevent stacking on re-source
10. Prompt (PS1): green user@host, blue cwd, git branch via __git_ps1
11. NVM setup
12. Atuin: eval "$(atuin init bash --disable-up-arrow --disable-ctrl-r)"
    Provides ghost-text autosuggestions via ble.sh integration; fzf/ble-bind own Ctrl+R and up arrow
13. ble.sh attach + fzf integration (MUST be last):
    ble-import fzf-key-bindings (Ctrl+R, Ctrl+T, Alt+C)
    ble-import fzf-completion (tab completion)
    ble-bind -m emacs -x up fzf-history-widget (up arrow → fzf history panel)
    ble-attach

INSTRUCTIONS FOR AI
====================
- Read this file before making any changes to the project.
- After every change, update the relevant section(s) of this file and bump
  the "Last updated" date at the top.
- Keep the .bashrc well-organized with numbered section comments.
- Do not add bloat — only include what the user explicitly asks for.
- Source order matters in .bashrc — be careful with PATH and env var ordering.
- command_not_found_handle (section 5) MUST remain before ble-attach (section 13). ble.sh wraps the pre-existing handler on attach; if defined after, it won't be wrapped.
- Do NOT use raw `bind -x` calls — ble.sh neutralizes them. Use `ble-bind` inside the `if [[ "${BLE_VERSION-}" ]]` block.
- Atuin MUST be initialized with --disable-up-arrow --disable-ctrl-r; ble.sh owns those bindings.
